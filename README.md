# pampu

CLI for Atlassian Bamboo.

## Installation

```bash
pip install pampu
```

## Setup

```bash
pampu init
```

Or set environment variables:

```bash
export BAMBOO_URL='https://bamboo.yourcompany.com'
export BAMBOO_TOKEN='your-personal-access-token'
```

## Usage

```bash
# List all projects
pampu projects
```
